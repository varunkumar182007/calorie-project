# Calorie Dataset Analysis

This project focuses on analyzing a calorie dataset using Python.

## 📊 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of nutritional values
- Classification of calorie levels (low, medium, high)

## 📁 Files
- calorie_dataset.csv → Dataset used
- proj.py → Main analysis code
- sample.py → Additional testing file

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
